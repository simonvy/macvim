Customize the MacVim.

Download: https://github.com/simonvy/macvim/blob/master/MacVim.dmg?raw=true

1. Add menu shortcuts for some plugins. git/janus is needed.
2. Add menu (Keys->open file) and the file search window. The window
filters the files in the current directory interactively. 
3. Auto hide the vertical scroller, as other OSX windows.
4. Remove the heading ~ in the empty lines.
